Computer Architecture and Assembly Language
This class focused on writing code in the Microsoft Macro Assembly (MASM) language. The final portfolio project focused on creating low-level I/O procedures. The program was able to read and write signed integers to the user through our own created procedures rather than using a library as we had been doing for the majority of the course. The source code for this project as well as a PDF outling the requirements for the project can be found on my Github.
